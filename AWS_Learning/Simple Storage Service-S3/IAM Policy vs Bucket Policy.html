IAM Policy vs Bucket Policy

This table compares the key differences between IAM Policies and Bucket Policies in AWS.

Feature IAM Policy Bucket Policy
Management Managed via the IAM dashboard Managed via the S3 dashboard
Format JSON JSON
Scope Applies to AWS services / Identity-based Applies to S3 buckets / Resource-based
Use Cases Broad AWS permissions Specific S3 permissions
Attached To IAM users, groups, or roles S3 buckets
Best For Multiple AWS services Specific S3 access
Cross-Account Complex configuration Ideal for S3 cross-account access
Public Access Not applicable Explicit public access control
