CloudFront Origin Settings

Understanding key configurations such as Origin Domain, Origin Path, Custom Headers, and Origin Shield.

Origin Settings Overview

Feature Description
Origin Domain Name The DNS name or endpoint from which CloudFront fetches content. Example: your-bucket.s3.amazonaws.com
Origin Path A directory or path within your origin which CloudFront uses as the root. Useful to point CloudFront to /static or /assets folders.
Add Custom Header CloudFront can send additional headers to your origin such as:
  • Authentication details
  • API keys
  • Debug metadata
  • Versioning information
Example use case: Sending an API key to backend servers.
Origin Shield A chargeable feature providing an additional caching layer that reduces load on your origin. Requests first pass through a centralized shield location, improving cache hit ratio.
