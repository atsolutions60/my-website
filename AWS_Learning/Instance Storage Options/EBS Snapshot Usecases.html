AWS EBS Snapshot Use Cases

# Use Case Purpose / Example
1 Periodic / On-Demand Backup Protect critical data. Example: Nightly snapshots of DB volumes to restore in case of corruption.
2 Move Data Across AZ / Region Enable redundancy or disaster recovery. Example: Copy snapshot from us-east-1a to us-west-2 and create new volume.
3 Restore Data After EC2 Deletion Attach volume from snapshot to new instance. Example: Recover web app after accidental instance deletion.
4 Enable Encryption Post-Creation Create encrypted volume from snapshot if original volume was unencrypted. Example: Secure sensitive data later.
5 Fast Snapshot / Quick Backup Quick incremental backup without impacting performance. Example: Instant snapshot for fast restore during updates or maintenance.
