AWS FSx Umbrella

FSx for Windows File Server

Fully managed SMB (CIFS) file storage for Windows workloads. Supports Active Directory integration, NTFS permissions, automatic backups. Ideal for corporate file sharing and Windows applications.

FSx for Lustre

High-performance parallel file system for HPC, big data, and ML workloads. Can link directly to S3 for fast data processing. Optimized for large-scale I/O and low latency.

FSx for NetApp ONTAP

Enterprise-grade storage with multi-protocol support (NFS, SMB, iSCSI). Features snapshots, replication, cloning, deduplication, and high availability. Great for enterprise applications and DR setups.

FSx for OpenZFS

Open-source ZFS file system with compression, snapshots, cloning, and data integrity. Supports high IOPS workloads and efficient backup/restore. Suitable for workloads requiring consistent performance.
