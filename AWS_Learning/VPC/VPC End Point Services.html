AWS VPC Endpoint Services

Introduction

Aspect Description
Definition A VPC (Virtual Private Cloud) endpoint service is like a special doorway that allows you to connect to certain online services in a secure and private way. Instead of going through the public internet, traffic flows securely within the AWS network.
Requirement Endpoint services require either a Network Load Balancer (NLB) or a Gateway Load Balancer (GWLB).

Before & After Endpoint Services

Key Considerations

Consideration Description
Region Specific An endpoint service is available in the Region where you created it.
Cross-Region Access You can access the endpoint service from other Regions using VPC peering.
Protocol Support An endpoint service supports traffic only over TCP.
